Origens da Programação em HTML

O HTML (HyperText Markup Language) surgiu no início da década de 1990, criado por Tim Berners-Lee, cientista da computação britânico, para estruturar documentos na web.

Inicialmente, o HTML era bastante simples, permitindo apenas a criação de documentos com texto e links. Com o tempo, evoluiu para incluir imagens, tabelas, formulários e muito mais.

Atualmente, o HTML é a base da web moderna, sendo constantemente atualizado pelo W3C (World Wide Web Consortium) para atender às necessidades das novas tecnologias.
